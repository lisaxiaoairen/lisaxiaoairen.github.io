####sublime快捷键
 1. Ctrl + / 注释
 2. Ctrl + 滚动　字体变大/缩小
 3. Ctrl + N 新建
 4. Ctrl + S 保存
 5. Ctrl + Z 撤销
 6. Ctrl + C 复制
 7. Ctrl + V 粘贴
 5. Ctrl + 回车 添加一行空行
 6. Ctrl + Shift +V 粘贴过程中保持缩进
 7. Alt + F3 选中选择的词
 8. Ctrl + W 关闭当前文档
 9. Ctrl + H 替换
 10. Ctrl + D 多行游标选择 可以搭配 Ctrl + K取消选择部分游标 产生游标的另外一种方式，按住Shift + 鼠标右键拖动光标
 11. Ctrl + Shift + D 复制这行文本
 12. SHift +Tab 去除缩进
 13. Ctrl + {/} 调节代码缩进
 14. Ctrl + shift + 上下箭头 上移或下移选中代码
